Camel Casing
  
 Previous Top Next
First character of all words, except the first word are upper case other characters are lower case. We will use camel casing in the following cases :

·     Variable names
      string firstName;

·     Parameters of methods
      int angleCounterClockWise;

·     For an interface 'I' with camel case name of interface.
            Ientity.
