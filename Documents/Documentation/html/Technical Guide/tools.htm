Various Operating systems, Softwares and tools used for the development of Internet Banking System are.

·     Ubuntu 8.04 - The host operating system for the whole development process.

·     Windows XP SP2 - For testing the portability on windows platform and the portability of web application in Internet Explorer version 6 and 7.

·     Eclipse IDE - For development process.

·     Rational Software Architect - For Object-Oriented design of the Software and Database Design of IBS.

·     IBM DB2 Universal Database - As database server for saving and maintaining the records of users and various transactions.

·     Websphere Application Server - The application server for deploying  and administrating the web application.

·     Subclipse - An Eclipse plugin for version control of the source code.

·     SVN - For subversion control and maintaining repository of the whole project.

·     HelpMaker 7.3.51.0 - For creating the compiled HTML (chm) help.

·     Tivoli Storage Manager - For creating and storing backup of the Database.
