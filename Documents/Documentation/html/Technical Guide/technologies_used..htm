Various Technologies used in the development are-

·     Java - The programming language chosen for the development of the Internet Banking System.

·     AJAX - Asynchronous Javascript and XML to send many requests to the server

·     JSP & Servlet - For Web Pages & Server End Scripting

·     Rich Internet Applications - For creating the rich look of interface & using Sandbox features for limiting access to resources. EXT-JS was used for this purpose

·     XML - Extended Markup Language

·     Web 2.0 Concepts
