Use Case: Print



                                                Fig-5.1.9 Print Use case.

Following are the specifications for this Use Case-

Actors:
·     Customers, Entrepreneurs, Industrialists, Organizations, Administrators, etc.

            Pre-Conditions
·     User logged in.

            Post-Conditions
·     None.


Basic Flow
·     The user will select a document to print.
·     User will be asked to print to printer/file or save as a file.
·     The control will be transferred to the server.
·     Server will send the document as a specific file format or to print.
·     The document will be printed/saved.
·     User will get a receipt.
