Use Case: Search



                                                Fig- 5.1.11 Search Use Case.

Following are the specifications for this Use Case-

Actors:
·     Customers, Entrepreneurs, Industrialists, Organizations, Administrators, etc.

Pre-Conditions
·     User is already logged in.

Post-Conditions
·     None

Basic Flow
·     The user will provide the search string and domain to be searched.
·     The request will be forwarded to the server.
·     The server will return the result.
·     Result will be displayed.
