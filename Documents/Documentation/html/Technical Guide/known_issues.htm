The known issues in the Internet Banking System are.

·    Once some data is loaded by AJAX on a particular request, on the next same or different AJAX request, data would be retrieved from the server. However,it will not be re-loaded on the grid. After refreshing the page, new data can be seen in the Grid. This is due to a bug in the API used.
