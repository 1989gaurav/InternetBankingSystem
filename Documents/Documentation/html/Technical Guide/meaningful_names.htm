Meaningful Names
  
 Previous Top Next
·     Use meaningful, descriptive variable name and don't use abbreviation.

String address ;                    // correct one
int angleCounterClockWise ;         // correct one
string add;                   // don't use abbreviation



·     Don't use single character like i, j, k rather than use index etc. single character can be used as iterators in loops.

·     UnderScore for variable name :
1.   Don't use underscore(_) for local variables.
2.   Use underscore(_) for member variable, so distinction between local and member variable will be more clear.

·     File name should match with the class name.

·     Use capital letters for constant variables.

·     Boolean variables should contain “is” prefix.

Boolean isValueChanged;
boolean _isDeveloped;
